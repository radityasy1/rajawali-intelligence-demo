### Implementation Notes:

The main app file (~8,900 lines) requires careful modification. Key areas to modify:

```python
# Database connection - Replace MySQL with SQLite
# OLD
DB_CONFIG = {
    'host': os.environ.get("DB_HOST"),
    'user': os.environ.get("DB_USER"),
    ...
}

# NEW
DB_PATH = os.path.join(os.path.dirname(__file__), "data", "demo.db")
```

```python
# LDAP auth - Replace with mock auth
# OLD
LDAP_SERVER = "ldap://10.250.193.116:389"
async def ldap_authenticate(username, password):
    ...

# NEW
async def mock_authenticate(username, password):
    VALID_USERS = {
        'demo': 'demo123',
        'reviewer': 'reviewer123',
        'test': 'test123'
    }
    if username in VALID_USERS and password == VALID_USERS[username]:
        return {'username': username, 'display_name': username.title()}
    return None
```
